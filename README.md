# Fraud Risk Score Engine

Project scaffold for "Fraud Risk Scoring Engine Using Mathematical Modeling + Explainable ML".

Structure:
- data/: raw and processed data + synthetic features generator
- src/: preprocessing, mathematical_model, ml_model, explainability, utils
- notebooks/: EDA and model development (placeholders)
- app/: Streamlit demo
- models/: saved artifacts
- results/: metrics and plots
